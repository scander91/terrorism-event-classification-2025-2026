================================================================
MINIMUM STANDARD FOR EACH SCRIPT
================================================================

Every .py file should have:

1. Module docstring at the top:
   """
   Chapter X: [description]
   This script [what it does].
   
   Usage:
       python script_name.py
   """

2. Imports organized:
   import os           # stdlib first
   import numpy as np  # third-party second
   from config import *  # local third

3. Functions with docstrings:
   def impute_missing(df, method="mean"):
       """Impute missing values using the specified method.
       
       Args:
           df: Input DataFrame
           method: "mean" or "median"
       
       Returns:
           DataFrame with imputed values
       """

4. Main block at the bottom:
   if __name__ == "__main__":
       main()

5. NO hardcoded paths — use config.py
6. NO API keys in code — use environment variables
7. NO print("debug123") — use proper logging or remove
